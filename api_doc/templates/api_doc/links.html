/api/shorten

Given a long URL, returns a utrim.me short-link.

Parameters

  • url - a long URL to be shortened (example: https://www.thenewboston.com).
  • format - json. Default: json.

Prerequisites

  • Long URLs should be URL-encoded. You can not include a longUrl in the request that has &, ?, #, or other reserved parameters without first encoding it.
  • Long URLs should not contain spaces: any longUrl with spaces will be rejected. All spaces should be either percent encoded %20 or plus encoded +. Note that tabs, newlines and trailing spaces are all indications of errors. Please remember to strip leading and trailing whitespace from any user input before shortening.

Return Values

  • shorturl - the actual utrim.me link that should be used, and is a unique value for the given utrim.me account.
  • url - an echo back of the url request parameter. This may not always be equal to the URL requested, as some URL normalization may occur (e.g., due to encoding differences, or case differences in the domain). This long_url will always be functionally identical the the request parameter.
  • created - the exact date and time when requested url got shortened.

Example Request


import requests

url="http://utrim.me/api/shorten"

data={'url':'https://www.geeksforgeeks.org/get-post-requests-using-python/'}
r = requests.post(url,data=data)
print(r.json())

Output


{'created': '2018-01-15 19:28:27.298391+00:00', 'shorturl': 'http://utrim.me/43u4o4', 'url': 'https://www.geeksforgeeks.org/get-post-requests-using-python/'}

 

/api/mylinks

  • A GET request results into a list of all the urls (alongwith respective meta data) contained in your url bank.
  • A POST request results into the supplied url being shortened and added to your url bank with a tag (which is also supplied in the POST request).

Authentication: Login Credentials

Parameters for POST

  • url - a long URL to be shortened (example: https://www.thenewboston.com).
  • tag - a name which may be associated to the url for future reference.

Prerequisites

  • Long URLs should be URL-encoded. You can not include a longUrl in the request that has &, ?, #, or other reserved parameters without first encoding it.
  • Long URLs should not contain spaces: any longUrl with spaces will be rejected. All spaces should be either percent encoded %20 or plus encoded +. Note that tabs, newlines and trailing spaces are all indications of errors. Please remember to strip leading and trailing whitespace from any user input before shortening.

Return Values

- For POST request

  • shorturl - the actual utrim.me link that should be used, and is a unique value for the given utrim.me account.
  • url - an echo back of the url request parameter. This may not always be equal to the URL requested, as some URL normalization may occur (e.g., due to encoding differences, or case differences in the domain). This long_url will always be functionally identical the the request parameter.
  • tag - an echo back of the tag request parameter.
  • created - the exact date and time when requested url got shortened.

- For GET request

  • Returns a list of url objects contained in your url bank. Each url object contains keys: 'shorturl', 'url', 'tag', 'created' with their respective values.

import requests

url="http://utrim.me/api/mylinks"

/api/mylinks/edit/<code>

  • A GET request retrieves the details of the url (contained in your url bank) when short-code corresponding to that url is supplied in place of <code> while making request.
  • A PUT request enables you to update the tag associated with the given url.
  • A DELETE request enables you to delete the given url object from your url bank.

Authentication: Login Credentials

Parameters for PUT

  • tag - a new name which you may want to associate to your url.

Prerequisites

  • <code> is to be replaced with the code string of length 6 which you can find at the end of your shortened utrim.me url (that url: whose details you want to get via this API call).

Return Values

- For GET request

  • shorturl - the actual utrim.me link that should be used, and whose 6 digit code was supplied in the requesting link.
  • original_url - That original long url which has been shortened to above utrim.me shorturl.
  • tag - name associated to this url in your url bank.
  • created - the exact date and time when original_url was shortened.

- For PUT request

  • Updates tag associated to the given url and returns the same details as for a GET request, but with the updated value of the tag.

- For DELETE request

  • Deletes the given url from your bank and returns nothing.

Example Request

- GET request


import requests

url="http://utrim.me/api/mylinks/edit/r361sb"

r = requests.get(url, auth=('username', 'password'))
print(r.json())

Output


{'original_url': 'https://www.geeksforgeeks.org/get-post-requests-using-python/', 'tag': 'req', 'created': '2018-01-15 20:49:10.298808+00:00', 'shorturl': 'http://utrim.me/r361sb'}

- PUT request


import requests

url="http://utrim.me/api/mylinks/edit/r361sb"
data={'tag':'req-resp'}
r = requests.put(url,auth=('username', 'password'),data=data)
print(r.json())

Output


{'tag': 'req-resp', 'created': '2018-01-15 20:49:10.298808+00:00', 'original_url': 'https://www.geeksforgeeks.org/get-post-requests-using-python/', 'shorturl': 'http://utrim.me/r361sb'}

- DELETE request


import requests

url="http://utrim.me/api/mylinks/edit/r361sb"

r = requests.delete(url,auth=('username', 'password'))
print(r.status_code)

Output


204

 

/api/mylinks/analytics/<code>

  • A GET request retrieves the detailed visit/hit analytics of the given url(contained in your url bank).

Authentication: Login Credentials

Parameters

  • None

Prerequisites

  • <code> is to be replaced with the code string of length 6 which you can find at the end of your shortened utrim.me url (that url: whose details you want to get via this API call).

Return Values

- For GET request

  • total_hit_count - the total number of hits on the given url till the current instant.
  • unique_ip_set - list of non-redundant/unique ip addresses that have visited the given link.
  • regionwise_unique_hits_percent - A dictionary/map having countries as keys and percentage of unique hits from these countries as corresponding values.

Example Request

- GET request


import requests

url="http://utrim.me/api/mylinks/analytics/pr43nc"
r = requests.get(url,auth=('username', 'password'))
print(r.json())

{

}
