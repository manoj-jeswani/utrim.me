Authentication

 

Most utrim.me API endpoints require Authentication credentials.

If you only need authentication credentials for your own account and will not be authenticating any additional end-users, you can generate one by signing up here.

 

Once you have login/authentication credentials, you can pass them like this :

auth=('username', 'password')

An example GET request using python might be made like this

import requests
url="http://utrim.me/api/mylinks"
r = requests.get(url, auth=('username', 'password'))
print(r.json())

Here, parameter auth is a tuple in which we supply registered account's username and password. Irrespective of request method ( get, post, put, delete ), auth will always be passed as a parameter with authentication credentials whenever request is being made to API endpoints which require user's authentication credentials before they can process the data flow.

An example request might look like:

GET /api/mylinks
